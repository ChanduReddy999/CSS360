position attributes are top/right/bottom/left........
relative:position ==== child elements follows the parent elements
position:static === by default all html elements will in static in position and attributes are not applied
position:fixed === is fixed at its position even if the position of other elements may be changed
position:absolute === is used for html elements such that it can be taken anywhere in the webpage as per the changes made in the css, it may go over the other elements while using the attributes
